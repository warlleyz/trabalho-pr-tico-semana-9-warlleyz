## Descrição
Projeto simples de um catálogo de produtos em formato de cards, utilizando JavaScript e manipulação do DOM.

## Funcionalidades
* Listagem de produtos
* Filtro por nome
* Filtro por categoria
* Exibição de detalhes do produto
* Destaque de produtos

## Tecnologias

* JavaScript
